# liri-node-app

<h1>About</h1>
<p>
Liri is a bot that I created as a tool for Language Interpretation and Recognition Interfacing. Liri processes commands and returns requested data to the user.

The user may use one of four commands:

1. concert-this
2. spotify-this-song
3. movie-this
4. do-what-it-says </p>
